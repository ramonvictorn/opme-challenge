# opme-challenge

- Development challenge using Node and React


# To run:
In code directory run:

`npm install  `
`npm install -D  `
`node src/server.js  `

And see on **localhost:4646**

# To run on docker:
` docker-compose up -d --build`

And see on **localhost:3005**

# Requirements 
Nodejs 8.10 or >
Docker and Docker-compose (optional if don't run on containers)
